# About

做此项目是为了将使用幕布网做的结构化笔记自动转换成anki的问答卡片


# 说明

>  现在只有一个简单的demo
>  先从幕布导出html文件放在项目根目录下
>  使用jupyter打开toanki.ipynb，第一行设置输入的html文件地址，最后一行设置输出的txt文件地址，从上往下运行，即可完成转换



## 技术栈
